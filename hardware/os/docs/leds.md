# LED Status

|                | Power                                 | Cloud                   | Active                |
| -------------- | ------------------------------------- | ----------------------- | --------------------- |
| Red            | Model in faulty state                 | Cloud Connectivity lost |
| Green          | FPGA Firmware loaded                  | Cloud Connectivity      | Experiment is running |
| Red Blinking   | Model in faulty state (unrecoverable) |
| Green Blinking |                                       | Connecting to Cloud     |
