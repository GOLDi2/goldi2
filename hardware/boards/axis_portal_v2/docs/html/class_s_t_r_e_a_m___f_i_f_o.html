Detailed Description

FIFO structure for temporary storage of stream data.

The STREAM_FIFO module is a dynamic First-In/First-Out memory structure designed for use in communication modules with external ICs. The fifo structure queues the outgoing data generated by other GOLDi submodules for to the IC, allowing the use of communication protocols slower than the custom GOLDi SPI protocol used by the main FPGA communication modules. Equally the fifo structure can be used to queue incomming data from the external IC until the GOLDi submodule can process or access the data.

The STREAM_FIFO uses a valid/ready handshake protocol to transfer data without loses. Data is registered and transfered only when both valid and ready flag are simultaneously asserted. Because of the nature of the protocol the fifo structure uses the input ready and output valid flags as negated "queue full" and "queue empty" flags.
