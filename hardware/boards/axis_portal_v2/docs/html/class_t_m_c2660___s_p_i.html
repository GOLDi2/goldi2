Detailed Description

Custom SPI master interface for TMC2660 (V3.00.00)

The TMC2660_SPI module acts as a custom master interface to control and configure the TMC2660 stepper motor driver IC. The module takes a 24-bit data word formatted as register data (see TMC2660 datasheet) and transfers it to the IC. The module also reads the read response of the IC and presents it in the read port "m_word_o_tdata".

The clock frequency for the SCLK serial clock can be configured using the "CLOCK_FACTOR" parameter, which gives the serial clock period as a multiple of clock cycles.
