Detailed Description

Actuator drivers and protection system the for GOLDI 3 Axis Portal (V1)

The module contains the drivers for the sensors and actuators of the GOLDI 3 Axis Portal (V1) system. The module constist of set of configuration registers accessed through a SPI port. Actuator signal are generated based on the register parameters and then fed into protection module that ensures the physical system isn't damaged. The error detector unit produces a list of errors for the user to analyse.
