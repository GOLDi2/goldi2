Detailed Description

DC Motor driver module for H-Bridge L293DD

H-Bridge driver for a DC motor with configurable registers The module is capable to drive the DC motor in both directions and at diffrent speeds by changin the PWM value and direction bits.

The PWM signal frequency can be configured through the CLK_FACTOR

CLK_FACTOR = (f_clk/f_pwm*255)

Registers:

Address Bit 7 Bit 6 Bit 5 Bit 4 Bit 3 Bit 2 Bit 1 Bit 0
+0 Out 1 Out 2
+1 PWM[7:0]
