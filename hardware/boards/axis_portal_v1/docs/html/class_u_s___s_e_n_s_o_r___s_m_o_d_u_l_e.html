Detailed Description

Ultrasonic sensor processing unit for HC-SR04.

The ultrasonic sensor dsp generates the signals necesary to drive the device and process the transmitted data. The module works by looping a measurement cycle every 60 ms and averaging the data using a moving average filter with the last 8 samples taken. The data formatted as a 16-bit unsigned value is stored in the module's registers.

A measurement cycle consists of a trigger burst, a echo measurement, and a wait period. The trigger bust holds the "p_us_trigger" high for 10 us to generate 8 ultrasonic pulses. Then the module enters an idle period and waits for a rising edge in the "p_us_echo" signal. Once the rising edge is detected a microsecond accumulator is increased until a falling edge in the "p_us_echo" singal is detected. This time corresponds to the time between the trigger burst and the detection of the echo. Finally the module enters a 20 ms idle state that prevents the detection of secondary echos generated by the first burst.

s[m] = (0,00034 m/us * t_echo[us])/2
