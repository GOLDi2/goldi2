Detailed Description

SPI to internal BUS standard.

Module acts as a SPI slave interface and a BUS master interface. The moduel manages the communication between the FPGA system and the microcontroller based on the configuration parameters set in the GOLDI_MODULE_CONFIG package (default: adr -> 7bit, dat -> 8 bit)

Data format:

Byte Bit 7 Bit 6 Bit 5 Bit 4 Bit 3 Bit 2 Bit 1 Bit 0
Byte 1 [in] we ADDRESS[6:0]
Byte 2 [in/out] DATA[7:0]
Byte + [in/out] DATA[7:0] -> ADDRESS+1
