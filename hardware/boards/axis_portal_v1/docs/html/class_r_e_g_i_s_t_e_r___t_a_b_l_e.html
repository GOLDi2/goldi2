Detailed Description

Configurable table of registers.

Table of registers with configurable base address, length and initialization values. BUS interface reads the internal port input data and writes to the internal output data registers while also flaging the internal system of the changes through the stb signals. The module uses assertions for verification of proper parameters configured in the GOLDI_MODULE_CONFIG package during synthesis.
