Detailed Description

Periodic signal frequency analyser.

The frequency analyser module samples the signal at a given rate to calculates the period of the input signal. The module uses the rising edges of the input signal to calculate the amount of samples in the period. The values are then averaged with the last four samples to clean the data. The data is formatted as 32-bit unsigned value
