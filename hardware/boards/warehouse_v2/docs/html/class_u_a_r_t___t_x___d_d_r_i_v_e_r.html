Detailed Description

UART transmitter module with configurable transfer rate.

The UART transmitter module takes an encoded data packet and transfer the information through the "p_tx" serial port. The data in the "p_dword_tdata" is registered in a shift regiser when the "p_dword_tready" and "p_dword_tvalid" signal are both asserted and then shifted to the serial port at a pre-configured data rate. To allow continous data transfers the "ready" signal is asserted before the ongoing transfer cycle is finised, after the last bit of the data packet begins, specifically at the begining of the last bit period.

When the module is in an idle state the transmission rate i.e. the sampling rate of the module can be selected from a list of common used values using the "drate" port. A simultaneous "tready" and "tvalid" high value transfers the data.

Transfer rates:

  • 000 - 9600
  • 001 - 19200
  • 010 - 38400
  • 011 - 57600
  • 100 - 115200
  • 101 - 230400
  • 110 - 460800
  • 111 - 921600
