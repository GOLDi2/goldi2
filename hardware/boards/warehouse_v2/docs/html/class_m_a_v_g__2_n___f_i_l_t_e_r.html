Detailed Description

Moving average filter.

The "MAVG_2N_FILTER" is a moving average filter module with a 2**N number of samples. The length of the data buffer is defined by the generic parameter "g_length_log" which is equal to N. The "p_sample_data" is an unsigned vector with a width defined by the parameter "g_data_width".
