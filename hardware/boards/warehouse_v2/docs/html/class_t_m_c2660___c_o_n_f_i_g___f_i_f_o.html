Detailed Description

Configuration loading module for initialization of TMC2660 (V3.00.00)

The TMC2660_CONFIG_FIFO is used to load the default configuration of the TMC2660 after reset or initialization. The default configuration is set through the "ROM" parameter, which consists of a list of 24-bit data words formatted as register data (see TMC2660 datasheet).

After the reset signal has been asserted the module enables the valid flag. A ready/valid handsake is needed for the data to be transfered. Once the list is empty the valid flag is grounded until a new reset occurs.
