Detailed Description

IO signals for use in the GOLDi system. The io_i and io_o types provide a standard interface for signals that are used in combination with tri-state buffers or the FPGA pins. The structures allow "in", "out", and "inout" signals to be routed accross the device and enables the GOLDi submodules to be statically and dynamicaly routed accross the FPGA pins without changes to the pin contstraints.
